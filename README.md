# HR Analytics Dashboard

This project focuses on analyzing key Human Resources metrics using **Excel**, **Power BI** ,**Tableau** and **MySql** to support data-driven decision making. The dashboards provide clear insights into employee attrition, workforce demographics, and performance trends.

## Tools Used
- Microsoft Excel
- Power BI
- Tableau
- MySQL

## Key Insights
- Employee attrition analysis
- Headcount and workforce distribution
- Demographic breakdown (age, gender, department)
- Performance and experience trends

## Files Included
- Excel dashboard file (.xlsx)
- Power BI dashboard file (.pbix)
- Tableau dashboard file (.twbx)
- MySQL text file
- All Dashboard screenshots for quick preview

## How to View
1. Download the Excel (`.xlsx`) or Power BI (`.pbix`) or tableau(`.twbx`) file from this repository.
2. Open the file using Microsoft Excel or Power BI Desktop or Tableau Public.
3. Use filters and slicers to explore insights interactively.

## Purpose
This project is created for **learning and portfolio demonstration**, showcasing practical HR Analytics skills using real-world style data and dashboards.
